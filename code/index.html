Code Lists Resgiter

The GNAF ontology uses 9 vocabularies for class instance classification. For example, the Number Types vocabulary classifies the various address numbers in the GNAF into First, Last, Lot, Level and Flat These vocabularies - think code lists - are:

  • Address Types
  • Alias Types
  • Flat Types
  • Geocode Types
  • Locality Types
  • Mesh Block Match Types
  • Number Types
  • Primary Secondary Join Types
  • Street Types
